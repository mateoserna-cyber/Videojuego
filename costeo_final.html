Costos No Considerados / Riesgos

  • GCP Network Egress — Trafico saliente de la VM (audio streams, API responses). Bajo volumen es free tier, pero a 100K+ llamadas podria sumar. ~$5-15 USD/mes
  • DIDs / Numeros telefonicos — El carrier SIP cobra $7/llamada pero no incluye arriendo de numeros DID. Si se necesitan numeros dedicados. ~$5-20 USD/mes
  • GCP Snapshots / Backups — Backups automaticos del disco de la VM Asterisk. Recomendado pero no incluido. ~$3-8 USD/mes
  • Dominio + DNS — Si se usa dominio propio para el dashboard/API. ~$1-2 USD/mes
  • Deepgram sin plan: rate limits — Pay-as-you-go tiene limites de concurrencia mas bajos. A alto volumen simultáneo podria requerir plan Growth igualmente. Riesgo operacional
  • Asterisk e2-standard-8 concurrencia — 8 vCPUs manejan ~80-120 llamadas simultaneas. Si el volumen se concentra en horario punta, podria necesitar escalar. Riesgo a >100K/mes
  • Logging / Observabilidad extra — Cloud Logging en GCP cobra por volumen de logs ingestados. Con debug activado puede sumar. ~$0-10 USD/mes
